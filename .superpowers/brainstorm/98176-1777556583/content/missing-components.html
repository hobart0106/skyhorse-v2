掃描結果：9 個遺漏 Pattern

掃描 index.html / ak_index.html / fleet.html / article.html / faq.html 後發現

建議納入設計系統
可選（頁面特定或低優先）
Stats Bar
建議納入
.stats-bar / .stat-item
橫排數字統計列，例：「10,000+ 租車次」「9年經驗」「4.9分」，首頁 hero 下方常見
出現：index.html、ak_index.html
FAQ Accordion
建議納入
.faq-item / .faq-question / .faq-answer
問答折疊元件，點擊展開/收合，多個頁面共用，有獨立的 faq.html 頁面
出現：index.html、faq.html、ak_index.html
Advantage Card
建議納入
.adv-card / .why-card
icon + 標題 + 描述的功能特色卡，用於「為什麼選我們」等區塊，多頁共用
出現：index.html、ak_index.html
Airport Card
建議納入
.airport-card
機場代號（SFO）+ 城市名 + 機場全名，網格排列，SkyHorse 核心服務展示
出現：index.html、fleet.html
Hero Booking Form
建議納入
.hero-booking-card / .hero-booking-form
Hero 區塊內嵌的預約表單卡（取車地點/日期/還車），是首頁最核心的互動元件
出現：index.html
Newsletter Section
建議納入
.newsletter
Email 訂閱橫幅，標題 + 副標 + input + 送出按鈕，可嵌入頁底或 section 之間
出現：index.html、blog.html
Sticky Bar
可選
.sticky-line
手機版滾動後出現的固定底部 CTA 列，行動裝置專用，轉換率工具
出現：ak_index.html
Article Layout
可選
.article-layout / .toc / .sidebar-cta
文章頁版面：主體內容 + 右側目錄（TOC）+ 側邊 CTA，僅 article.html 使用
出現：article.html
Breadcrumb
可選
.breadcrumb
路徑導覽列，首頁 › 阿拉斯加 › 夏季行程，SEO 與導覽用，目前僅少數頁面有
出現：article.html、ak_index.html
