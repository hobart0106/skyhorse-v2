Color Token 架構 — Dark / Light Mode 支援

三層架構：原色盤 → Semantic Token → 元件

架構設計
Layer 1 — Primitive
原始色值，不直接用於元件
--primitive-gold-400: #FFDC22
→
Layer 2 — Semantic
有意義的角色名，theme 切換在這層
--color-accent: var(--primitive-gold-400)
→
Layer 3 — Component
元件只引用 semantic token
.btn-primary { background: var(--color-accent) }
