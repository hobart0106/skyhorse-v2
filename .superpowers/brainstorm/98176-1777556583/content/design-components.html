Section 2 — 核心元件

6 個元件 + 1 個 Badge utility，全部使用 token 變數

01 Navigation .nav
SkyHorse
車隊服務範圍關於我們
半透明黑底 + blur(20px)，高度 64px，CTA 使用 --gold，scroll 後加底部邊框
02 Button .btn .btn-primary / .btn-secondary / .btn-ghost
Primary: --gold 底色，Secondary: 透明 + border，Ghost: 最低調。加 .btn-sm 縮小版
03 Card .card
[ 車輛圖片 ]
Toyota Camry
中型轎車 · 5人座 · 自動
$45/天
hover: translateY(-4px) + gold glow shadow，圓角 --r-lg (16px)，底色 --bg-card
04 Chip & Tag .chip / .chip-tag
全部 轎車 SUV 豪華    自動擋 5人座 SFO可取車
Chip 用於篩選器（可選中狀態 .active），Tag 用於車輛屬性標籤，圓角 --r-pill
05 Form Input .form-group .form-label .form-input
例：SFO、LAX、JFK
請選擇有效日期
focus: --gold 邊框，error: 紅色邊框 + hint，底色 --bg-elevated，圓角 --r-md (10px)
06 Badge .badge .badge-gold / .badge-success / .badge-info
熱門 有庫存 新上架
狀態標記用，三種語義色，可嵌入 Card 或列表中
